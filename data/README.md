# Data Notes

The full experimental datasets are not bundled in this repository.

## Why

- The datasets used for reproduction are typically much larger than what makes sense to commit into a small class-project repo.
- Some datasets may have separate licenses, download agreements, or preprocessing steps.

## What To Add Later

- The exact dataset chosen for the first reproduction target
- Where it comes from
- Any download or access instructions
- Any preprocessing or train/test split notes

## Likely Starting Point

Based on the upstream Narcissus README, a practical first path may involve:

- CIFAR-10 as the target dataset
- Tiny ImageNet as auxiliary data in the default notebook workflow

Create subfolders here later if needed, for example:
